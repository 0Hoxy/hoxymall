**2024-10-13**
- CRUD
- SKU 8~12자리 까지 가능한 난수 생성

**2024-10-14**
- 카테고리 엔티티 생성
- 카테고리와 상품 연관관계 매핑
- 카테고리 CRUD

**2024-10-15**
